# Quiz App

## Project Description
This is a responsive Quiz App built using HTML, CSS, and JavaScript. It fetches multiple-choice questions from the Open Trivia Database API and displays them to the user.

## Features
- Fetches quiz questions using Fetch API
- Responsive design for mobile and desktop
- Highlights correct and incorrect answers
- Loads a new question on button click

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## How to Run
1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Open `index.html` using Live Server.

## API Used
Open Trivia Database
https://opentdb.com/
